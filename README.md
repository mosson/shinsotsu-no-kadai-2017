# This is Study Project

## Usage

### 1. install node.js

`https://nodejs.org/ja/` or `homebrew` or etc..

### 2. export bin path

open terminal.app, then type below.

```
$ export PATH=$PATH:./node_modules/.bin
```

### 3. install libs

```
$ npm i
```

### 4. start dev

```
$ npm run dev
```

enjoy.
